Application for Gachon university students

In 2020, we created an application called Gachon Handbook to ease finding information related to university. As most web applications are in korean language, it is difficult for foreigners to get information

This web application was created in in MIT App Inventor website. I found it a great website for creating applications. There is no need to write any code there which makes it easy but because of this, its scope is limited and cannot compete with other programming languages.

Overall, it is a small group project and 3 of us who are stutying in computer engineering did it in a short period of time. It is available in Play store and you may download it from here.
